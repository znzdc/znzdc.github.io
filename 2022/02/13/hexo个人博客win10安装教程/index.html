hexo个人博客安装教程（win10）

1.下载安装node.js

  • 在浏览器输入网址：https://nodejs.org/en/

  • 选择如下图16.14.0下载

![Alt text]

  • 下载后直接点下一步安装

  • node -v #查看node版本

npm -v #查看npm版本

2.下载安装git

  • 在浏览器输入网址：
    https://npm.taobao.org/mirrors/git-for-windows/

  • 选择自己需要的版本下载

  • 下载后直接点下一步安装

  • 安装后在空白处右击选择Git Bash Here

  • 在终端输入
    git config –global user.name ‘你自己注册github的昵称’
    git config –global user.email ‘你自己注册github的邮箱’

    3.安装淘宝的cnpm 管理器

    • 在空白处右击选择Git Bash Here
    • 输入 npm install -g cnpm –registry=http://registry.npm.taobao.org 回车等待安装完成
    • cnpm -v #查看cnpm版本

      4.安装hexo框架

  • 在空白处右击选择Git Bash Here

  • 输入 cnpm install -g hexo-cli 回车等待安装完成

  • hexo -v #查看hexo版本

  • 创建blog目录

  • 进入blog目录

  • 在空白处右击选择Git Bash Here

  • 输入 hexo init 按回车 #生成博客 初始化博客

  • hexo s #启动本地博客服务

  • http://localhost:4000/ #本地访问地址

5.将博客部署到gitHub

  • cnpm install –save hexo-deployer-git #在blog目录下安装git部署插件
  • 在自己的gitHub新建仓库仓库名必须为：昵称+github.io
  • 由于github在2021年8月13日后不再使用密码方式验证身份，现在使用个人token
  • 生成个人token方法链接：https://blog.csdn.net/chengwenyang/article/details/120060010
  • 修改blog目录下的_config.yml文件
